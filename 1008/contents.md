# A/B

|시간 제한|메모리 제한|
|:--------|:----------|
|2 초|128 MB|

### 문제
A/B를 계산하시오.

### 입력
첫째 줄에 A와 B가 주어진다. (0 < A,B < 10)

### 출력
첫째 줄에 A/B를 출력한다. 절대/상대 오차는 10^-9 까지 허용한다.

#### 예제 입력

```
1 3
```

#### 예제 출력

```
0.33333333333333333333333333333333
```

#### 예제 입력

```
4 5
```

#### 예제 출력

```
0.8
```
