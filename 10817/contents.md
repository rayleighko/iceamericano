# 세 수
 
|시간 제한|메모리 제한|
|:--------|:----------|
|1 초|256 MB|

### 문제
세 정수 A, B, C가 주어진다. 이 때, 두 번째로 큰 정수를 출력하는 프로그램을 작성하시오. 

### 입력
첫째 줄에 세 정수 A, B, C가 공백으로 구분되어 주어진다. (1 ≤ A, B, C ≤ 100)

### 출력
두 번째로 큰 정수를 출력한다.

#### 예제 입력
```
20 30 10
```

#### 예제 출력
```
20
```

#### 예제 입력
```
30 30 10
```

#### 예제 출력
```
30
```

#### 예제 입력
```
40 40 40
```

#### 예제 출력
```
40
```

#### 예제 입력
```
20 10 10
```

#### 예제 출력
```
10
```
